# Tushare Pro 注册指南

## 🚀 快速注册（5分钟搞定）

### 步骤1：注册账号
1. 访问：https://tushare.pro/register
2. 填写邮箱和密码完成注册
3. 激活邮箱验证链接

### 步骤2：获取 Token
1. 登录后进入个人中心
2. 在首页即可看到您的 **Token** (一长串字符)
3. 复制这个 Token

### 步骤3：在系统中使用
有三种方式设置 Token：

#### 方式1：在界面中输入（推荐新手）
- 在 Streamlit 界面左侧的 "Tushare Token" 输入框粘贴即可

#### 方式2：设置环境变量（推荐熟练用户）
```bash
# Mac/Linux
export TUSHARE_TOKEN="你的token"

# Windows
set TUSHARE_TOKEN=你的token
```

#### 方式3：创建 .env 文件
在项目根目录创建 `.env` 文件：
```
TUSHARE_TOKEN=你的token
```

## 🎯 与 AkShare 对比

| 特性 | Tushare Pro | AkShare |
|---|---|---|
| 数据质量 | ⭐⭐⭐⭐⭐ 官方专业数据 | ⭐⭐⭐ 爬虫数据，可能不稳定 |
| API 稳定性 | ⭐⭐⭐⭐⭐ 官方长期维护 | ⭐⭐ 第三方接口常变更 |
| 速度 | ⭐⭐⭐⭐ 较快 | ⭐⭐ 有时很慢 |
| 数据完整性 | ⭐⭐⭐⭐⭐ 无缺失 | ⭐⭐⭐ 偶尔有 Open=0 等问题 |
| 使用门槛 | 需要注册 Token | 无需注册 |

**结论**：强烈推荐使用 Tushare Pro！注册仅需5分钟，但数据质量提升巨大。
