# 新功能使用指南

## 🎉 新增功能总览

基于 AkShare 数据源，我们新增了3大核心功能：

### 1. 🔥 板块热点分析
**功能**：
- 实时概念板块排行（441个概念）
- 行业板块排行（86个行业）
- 市场情绪分析（涨跌统计）
- 涨停板追踪

**价值**：
- 发现市场热点
- 找到领涨板块和龙头股
- 把握投资方向

### 2. 💰 龙虎榜追踪
**功能**：
- 最近N天龙虎榜数据（1-7天可选）
- 主力资金进出统计
- 机构席位活动分析
- 上榜原因分类

**价值**：
- 追踪主力资金动向
- 发现异动股票
- 辅助投资决策

### 3. 📰 个股新闻 & 市场要闻
**功能**：
- 个股相关新闻（最新10条）
- 市场重要新闻
- 实时更新

**价值**：
- 及时了解个股动态
- 把握市场节奏
- 避免重大风险

## 🚀 使用方法

### 方法1：独立页面（推荐）

**启动市场热点页面**：
```bash
./启动市场热点.command
```

**功能**：
- 📊 板块热点：概念排行 + 行业排行 + 市场情绪
- 💰 龙虎榜：最近N天数据 + 详细记录
- 📰 市场要闻：最新财经新闻

**优点**：
- 独立运行，不影响个股分析
- 专注于宏观市场
- 界面清晰，数据完整

### 方法2：命令行测试

**快速测试**：
```bash
./venv/bin/python test_new_features.py
```

**输出**：
- 板块热点前5名
- 龙虎榜统计
- 新闻标题列表

**适用**：
- 快速查看数据
- 脚本化集成
- 自动化监控

### 方法3：集成到主应用（待完善）

在主应用的侧边栏会显示：
- 🔥 今日热点（概念板块前5）
- 💰 龙虎榜（今日上榜股票数）

在个股分析结果下方会显示：
- 📰 相关新闻（该股票最新新闻）

## 📊 数据说明

### 板块数据
- **数据源**：东方财富
- **更新频率**：实时
- **板块数量**：441个概念 + 86个行业
- **指标**：板块名称、涨跌幅、领涨股票、成交额等

### 龙虎榜数据
- **数据源**：东方财富
- **数据范围**：最近7天
- **包含信息**：股票代码/名称、上榜日期、涨跌幅、上榜原因、买卖席位、资金数据等

### 新闻数据
- **数据源**：东方财富
- **更新频率**：实时
- **数据量**：个股新闻10-20条，市场要闻20条
- **包含信息**：标题、发布时间、新闻内容

## 💡 使用技巧

### 技巧1：找热点股票
1. 打开"市场热点"页面
2. 查看"概念板块排行"
3. 点击涨幅最大的板块
4. 查看"领涨股票"
5. 在"个股分析"中深入分析

### 技巧2：追踪主力
1. 打开"龙虎榜"tab
2. 筛选净买入为正的股票
3. 查看上榜原因和席位
4. 结合个股分析判断趋势

### 技巧3：避免踩雷
1. 分析个股前先看新闻
2. 关注负面消息
3. 结合龙虎榜看主力动向
4. 谨慎决策

## 🔧 技术架构

### 核心模块
```
stock_analysis/
├── analysis/
│   ├── market_hotspot.py    # 板块热点分析器
│   └── dragon_tiger.py       # 龙虎榜分析器
├── data/
│   └── news_provider.py      # 新闻提供者
└── ui/
    └── market_hotspot_page.py  # 热点页面UI
```

### API接口（AkShare）
```python
# 板块
ak.stock_board_concept_name_em()  # 概念板块
ak.stock_board_industry_name_em() # 行业板块

# 龙虎榜
ak.stock_lhb_detail_em(start_date, end_date)

# 新闻
ak.stock_news_em(symbol)  # 个股新闻
```

## ⚠️ 注意事项

1. **网络依赖**：
   - 所有数据来自网络，需要稳定的网络连接
   - 首次加载可能较慢（2-5秒）

2. **数据延迟**：
   - 龙虎榜：T+1（当天数据次日公布）
   - 板块数据：实时
   - 新闻：实时

3. **API限制**：
   - AkShare无明确限制
   - 但建议避免高频调用
   - 间隔2-3秒为佳

4. **数据准确性**：
   - 数据来自公开渠道
   - 仅供参考
   - 投资需谨慎
