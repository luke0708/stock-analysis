## 1. 🎉 核心功能总览

本系统整合了 **AkShare** (免费) 和 **Tushare Pro** (专业) 双数据源，提供以下核心价值：

### 1.1 📈 个股全景分析
- **资金流向**：全天累计净流入、主力/散户博弈分析
- **热力图谱**：日内10分钟级别的资金活跃度热力图
- **买卖力度**：通过买卖盘挂单差异分析多空力量
- **多股对比**：双股票走势与资金流叠加对比 (Pro功能)

### 1.2 🔥 市场热点追踪
- **板块排行**：实时概念/行业板块涨跌幅排行
- **龙虎榜**：追踪游资和机构的席位动向
- **市场情绪**：全市场涨跌家数统计

### 1.3 🛠 实用工具箱
- **数据导入/导出**：支持 CSV 离线分析
- **自选股管理**：本地持久化保存关注列表
- **智能缓存**：流畅的交互体验，自动管理内存

## 3. 📊 深度功能说明

### 3.1 📥 数据导入与离线分析
支持上传之前保存的 CSV 文件进行回溯分析。
1. 在侧边栏点击 **"⚙️ 高级设置"**。
2. 选择 **"导入CSV文件"**。
3. 上传文件后点击 **"导入并分析"**。
*适用场景：复盘历史行情、对比不同日期的资金流向。*

### 3.2 ❓ 指标说明系统
遇到不懂的专业术语？
1. 点击侧边栏的 **"❓ 指标说明"**。
2. 选择指标（如 VWAP、RSI 等）。
3. 系统将显示定义、计算公式和实战解读。

### 3.3 🗑️ 缓存管理
系统会自动缓存数据以提升速度。如果遇到内存不足或数据未更新：
1. 点击侧边栏 **"⚙️ 系统管理"**。
2. 点击 **"🧹 一键清理"**。

## 4. 📈 图表与算法原理

本节详细说明核心图表的设计原理，助您看懂数据背后的逻辑。

### 4.1 全天累计资金流曲线
- **图表**: 红色面积图。
- **含义**: 每一分钟的净流入额累加。曲线向上代表多头主导，向下代表空头抛压。
- **算法**: `累计净流入[t] = 累计净流入[t-1] + (买盘额 - 卖盘额)`

### 4.2 日内分时资金流热力图
- **图表**: 红蓝配色的柱状/色块图。
- **含义**: 展示每10分钟的资金流强度。
  - **红色**: 净流入（颜色越深流入越多）
  - **蓝色**: 净流出
  - **高度**: 资金流比率（净额/成交额）
- **价值**: 快速定位全天资金最活跃的时间窗口。

### 4.3 主力/散户资金流构成图
- **图表**: 堆叠面积图。
- **含义**: 分离“主力”和“散户”的资金动向。
- **智能标注**:
  - 🔴 **主力吸筹**: 主力大幅流入而散户流出（买入信号）。
  - 🟢 **主力出货**: 主力大幅流出而散户流入（卖出信号）。

### 4.4 已知限制
- **主力/散户划分**: 目前基于分钟数据的成交量估算（1.5倍均量）。后续版本接入 Tick 数据后将更加精准（>20万/笔）。

## 5. 🔧 数据源配置 (Tushare)

本系统默认使用 **AkShare (免费)**，也支持 **Tushare Pro (高质量)**。

### 为什么使用 Tushare Pro?
- **更稳定**: 官方接口，极少失败。
- **更完整**: 历史数据无缺失。

### 配置步骤 (5分钟)
1. **注册**: 访问 [Tushare Pro](https://tushare.pro/register) 注册账号。
2. **获取 Token**: 登录后在个人中心复制 Token。
3. **配置**: 
   - 方式 A: 在系统侧边栏输入框直接粘贴。
   - 方式 B: 在项目根目录创建 `.env` 文件：
     ```
     TUSHARE_TOKEN=你的token
     ```

## 6. 💡 常见问题 (FAQ)

**Q: 数据会积累导致电脑变慢吗？**
A: 不会。系统采用 Session 缓存，关闭网页或重启应用后会自动释放内存。

**Q: 为什么有时候看不到“龙虎榜”数据？**
A: 龙虎榜数据通常在交易日 16:30-17:30 之后公布。盘中无法查看当日榜单。

**Q: 多股对比页面的两条线代表什么？**
A: 实线代表“左轴”股票，虚线代表“右轴”股票。双轴设计是为了解决两只股票价格或成交额量级差异过大的问题。
