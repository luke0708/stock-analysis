# A股资金流向智能分析系统 - 用户手册

本文档面向最终用户，重点说明如何使用各页面与功能。

## 1. 快速开始

### 方法 A：双击启动（macOS 推荐）
在项目根目录双击 `启动分析系统.command`，然后打开浏览器访问 `http://localhost:8501`。

### 方法 B：命令行启动
```bash
streamlit run stock_analysis/ui/unified_app.py
```

---

## 2. 页面导航与核心操作

### 2.1 市场全景
- 展示上证/深证/创业板等指数概览。
- 行业板块涨跌与板块分析（含成分股拉取耗时/失败提示）。
- 市场要闻与热点资讯。

### 2.2 个股资金流向分析（核心）
1. 输入股票代码，选择日期，点击“开始分析”。
2. 系统会自动显示**实际分析日期**（若所选日无数据，会回退到最近交易日）。
3. 重点图表包括：分时走势、资金流热力图、主力/散户结构、大单追踪等。
4. 页面底部提供图表 AI 解读（独立于智能投顾对话）。

**提示**：
- 免费数据源通常只覆盖近 1 周分钟数据。
- 若提示“未能获取数据”，建议选择更近的交易日。

### 2.3 多股对比
- 同时输入两只股票代码与日期。
- 走势叠加与资金流对比图用于观察强弱差异。

### 2.4 实时预警
- 支持多股监控列表、连续触发与阈值设定。
- 自动判断交易时段，非交易时段默认暂停刷新。
- 建议刷新间隔 ≥ 30 秒，监控数量 ≤ 5。

### 2.5 自选股（可选）
- 本地保存关注列表，便于快速切换分析对象。

### 2.6 全球市场
- A 股指数快照 + 美股/港股（YFinance）。
- 页面会展示数据源与耗时信息。

### 2.7 AI 智能投顾
- 需先完成一次“个股资金流向分析”，再进入 AI 页面。
- 可选择解读侧重点与风格，支持追问。
- **新闻补充（可选）**：勾选“包含最新相关新闻”后拉取数据，模型才会基于新闻回答问题。

---

## 3. 数据源说明

- **AkShare（默认）**：免费、覆盖广，但分钟数据时效有限。
- **Tushare（可选）**：更稳定，需在 `.env` 中配置 `TUSHARE_TOKEN`。
- **YFinance**：全球市场数据来源。
- **新闻来源**：AkShare 相关新闻，可能存在延迟或缺失。

---

## 4. 常见问题（FAQ）

**Q1：为什么日期会自动回退？**
A：所选日期无交易数据时，系统会自动切换到最近交易日，并在页面顶部显示实际日期。

**Q2：为什么 AI 说无法判断新闻影响？**
A：未勾选“包含最新相关新闻”时，模型不会使用新闻数据。请先拉取新闻再提问。

**Q3：大单占比是否准确？**
A：分钟级数据为聚合数据，大单占比是近似估算；若未来接入 Tick 数据会更精准。

**Q4：自动刷新没反应？**
A：非交易时段会自动暂停刷新，需手动刷新或等待交易时段。
